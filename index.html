O que é um Servidor?

Bom, explicando de forma simples, servidor é um computador com uma grande capacidade de processamento e armazenamento comparado ao computador comum que a maioria das pessoas tem em suas casas hoje em dia. A função dele é centralizar dados/informações, criando serviços (Contêineres, grupos, usuários) na rede e aplicando leis (GPOs) restringindo acessos a esses dados. Diferença de máquinas:

  • Servidor: Grande processamento e armazenamento de dados, administrador da rede, distribui serviços;
  • Cliente: Pequeno processamento e armazenamento de dados, usuário da rede, acessa serviços;

Existem diversos serviços que são distribuídos de um servidor para uma rede de computadores. Principais serviços e mais usados são:

  • Dynamic Host Configuration Protocol (DHCP): Distribui os IPs na rede a partir da primeira vez que um computador se conecta a rede;
  • Domain Name System (DNS): Identifica e traduz os IPs na rede;
  • Active Directory (AD): Cria Contêineres, Grupos, Usuários e faz o gerenciamento deles na rede;

Somente com esses recursos (dentre muitos outros) é possível criar uma rede de computador eficaz.

Principais Servidores usados

  1. Arquitetura de cliente & servidor

    2. Permite o gerenciamento (distribuição) de tarefas na rede. Faz o roteamento centralizado de informações.

  2. Computação em nuvem

    3. Nos dias de hoje, um dos principais conceitos ligados a servidores é a computação em nuvem. Uma pessoa não precisa mais comprar um servidor, ela pode usar a nuvem a favor dela, pagando apenas o que usou. Isso favorece em muitos aspectos, como:

    • Paga somente o que usar;
    • Não precisa pagar manutenção;
    • Distribuição de recursos de forma ilimitada, sem problema de latência ou perda de dados;
    • Disponibilidade de uma grande quantidade de serviço;

    Muitas empresas (de pequeno e médio porte) utilizam computação em nuvem para o uso de servidores, porque como o uso não é de grande escala, o preço fica bem em conta.

  3. Alojamento Web

    4. É normal utilizar servidor para a hospedagem de sites, e na maioria das vezes, muitas pessoas não tem um potencial financeiro para comprar um servidor, e somente para alocar um site. Por esse caso e alguns outros que surgiram os alojamentos web. São empresas que fornecem acesso restrito ao servidor deles, permitindo que o usuário aloque o site dele nesses servidores. Hoje em dia esse método é muito utilizado.

Porque utilizar um Servidor?

Com os servidores, todos tipos de informações do seu comércio ficam gravadas nele, para que com o gerenciamento administrativo, o acesso dessas informações seja restrito somente para quem tem acesso (usuários, grupos). Um servidor fornece segurança a rede com seu firewall, protegendo seus dados, e evitando a perda geral de informações.

Como instalar um Servidor

Vamos aprender a instalar um Windows server, com sua máquina acoplada com um sistema Windows Server, faça as seguintes instruções.

  • Escolha o idioma da máquina;
  • Leia e aceite os acordos do sistema;
  • Escolha o modelo do seu sistema. Obs.: As opções sem o “(Desktop Experience)” são sistemas sem gráfico, funciona somente por linhas de comando;
  • Escolha o particionamento personalizado;
  • Siga para próxima etapa caso queira somente aprender a administrar o servidor, se não, particione conforme a sua necessidade;
  • Instale e espere;
  • Adicione sua senha, Obs.: O username da máquina sempre vai ser Administrador ou Administrator se estiver em inglês, esse é um padrão os servidores Windows;

A partir deste ponto o usuário já terá acesso ao Service Manager e poderá começar a criar os serviços.
